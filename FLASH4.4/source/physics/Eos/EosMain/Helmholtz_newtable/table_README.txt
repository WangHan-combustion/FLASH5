The table contains the following information in the following format

Row 1: number of variables	number of derivatives
Row 2: number of cells computed in each direction
Row 3: Flags for whether log is used in each direction
Row 4: density array size for this table	density array giving cell boundaries
Row 5: energy array size for this table 	energy array giving cell boundaries
Row 6: Ye array size for this table		Ye array giving cell boundaries
Row 7: sumy array size for this table		sumy array giving cell boundaries

Row 8-16:
Columns 0-48: the weights for entropy and two first derivatives 
Columns 49-52: the four hyperparameters  L1, L2, L3, L4, for energy, density, Ye, and sumy respectively
